Hallo om pengguna Source Code Aliendroid, Akhir-akhir ini banyak sekali aplikasi yang mendapatkan peringatan terkait pengisian kemanan data / data safety. Keamanan data akan diwajibkan pada bulan April 2022, sehingga seluruh aplikasi/game yang terdeteksi terkait adanya pengumpulan data wajib untuk melakukan update. Merujuk pada artikel Data Safety, maka penulis coba uraikan pengisian sesuai sc alien atau non alien dengan melihat poin berikut:

1. Dasar pengisian form

Untuk mengisi form data safety, penulis merujuk pada bagian AndroidManifest.xml (penggunaan permisi) dan SDK/Libs apa saja yang digunakan sesuai keterangan "Provide information for Google Play's Data safety section". Untuk kasus sc Alien maka permintaan permisi hanya ijin baca dan tulis file kedalama memory Smartphone.

Meskipun pada bagian AndroidManifest.xml tidak dituliskan pelacakan user sesuai lokasi, akan tetapi SC alien tetap melacak lokasi pengguna dikarenakan penggunaan SDK iklan (Admob, dll) dan firebase. Ijin pelacakan  terdapat di sdk yang digunakan.
